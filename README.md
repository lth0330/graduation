# 아파트 주차 관리 시스템 (Graduation Project)

## 프로젝트 개요
- 목적: 아파트 주차 관리 업무를 웹/앱/카메라 연동으로 통합 관리
- 현재 범위: **웹 개발(관리자/아파트 관리자/주민 연계 기능)**
- 아키텍처: Frontend(React) + Backend(Spring Boot, Java) + MySQL
- 인증: JWT 기반 인증/인가

## 역할별 기능

### 1) Web 관리자
- 아파트 관리자 회원가입 신청 목록 조회
- 아파트 관리자 승인/거절
- 승인 시 아파트 정보 생성(아파트명 기반)
- 승인 시 6자리 랜덤 아파트 비밀번호 생성

### 2) Apartment Manager (아파트 관리자)
- 회원가입 신청(아이디, 비밀번호, 아파트명 등)
- 승인 후 로그인 가능
- 마이페이지에서 아파트명/아파트 비밀번호 확인
- 주민 회원가입 신청 목록 승인/거절
- 주민 정보 CRUD
- 주민 차량 정보 CRUD
- 주차장 정보 CRUD
- 주차 구역 정보 관리

### 3) Resident (아파트 주민)
- 회원가입 신청(아이디, 비밀번호, 아파트 비밀번호, 동/호수, 차량 정보)
- 아파트 비밀번호 검증 후 아파트 연결
- 승인 후 서비스 이용

## 추가 기술 요구사항 반영 계획
- 비밀번호 보안: bcrypt 해시 저장(평문 저장 금지)
- 주소 검색: Kakao Address API 연동
- 이메일 전송: Gmail API로 승인/거절 결과 안내

## 폴더 구조
- `backend/`: Spring Boot API 서버
- `frontend/`: React 웹 클라이언트
- `docs/`: ERD/API/화면설계 등 문서

## 개발 예정 기능
- JWT 로그인/리프레시 토큰 정책
- 역할 기반 접근 제어(RBAC)
- 승인 상태 워크플로우(pending/approved/rejected)
- 주민/차량/주차장/주차구역 관리 화면 및 API
- 이메일 템플릿 및 발송 이력 관리
- 주소 검색 UI/백엔드 연동

## 실행 준비(예정)
- Backend: Java 17+, Gradle, MySQL
- Frontend: Node.js LTS, npm
